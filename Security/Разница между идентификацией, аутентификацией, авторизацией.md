## Разница между идентификацией, аутентификацией, авторизацией?

**Разница между идентификацией, аутентификацией и авторизацией:**  

1. **Идентификация** - определение пользователя по уникальному признаку.  *"Кто ты?"*.

2. **Аутентификация** - проверка подлинности пользователя с помощью учётных данных.  *"Докажи это"*. 

3. **Авторизация** - предоставление пользователю прав доступа к определённым ресурсам или функциям после успешной аутентификации.  *"Что тебе разрешено?"*.
