## Что такое REST API?

**REST API** - это программный интерфейс, реализующий принципы REST, когда все данные рассматриваются как ресурсы, каждый из которых представлен уникальным унифицированным идентификатором ресурса (URI), позволяющий клиентам взаимодействовать с сервером через HTTP, передавая данные.
